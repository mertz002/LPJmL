NAME
          pnet_hi - Get upper bound for local sublattice

     SYNOPSIS
          #include <mpi.h>
          #include "pnet.h"

          int pnet_hi(const Pnet *pnet);

     DESCRIPTION
          Function returns upper bound for local sublattice. Function
          is implemented as a macro.  The argument of pnet_hi is:

          pnet Pointer to initialized Pnet struct

     RETURN VALUE
          Upon successful completion upper bound is returned.
