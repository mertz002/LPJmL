NAME
          freeconfig - Deallocate memory for LPJmL configuration

     SYNOPSIS
          #include "lpj.h"

          void freeconfig(Config *config);


     DESCRIPTION
          Function deallocates memory for LPJmL configuration.  The
          argument of freeconfig is:

          config
               LPJmL configuration data.

     RETURN VALUE
          None.
