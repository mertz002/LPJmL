     NAME
          printheader - Print header of clm files for LPJmL

     SYNOPSIS
          printheader clmfile ...

     DESCRIPTION
          Program prints file header of clm data files for LPJmL.

     OPTIONS
          clmfile filename of clm data file(s)

     EXAMPLE
          Print headers of all clm files in the current directory:
               printheader *.clm

     EXIT STATUS
          printheader returns a zero exit status if header of clm file
          could be read.  Non zero is returned in case of failure.
