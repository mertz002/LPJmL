headersize.1


     NAME
          headersize - print header size of clm files

     SYNOPSIS
          headersize [-version v] clmfile ...

     DESCRIPTION
          File header of clm files is read and size of header in bytes
          printed.

     OPTIONS
          -version v
               version of clm file set to v, by default version is
               read from file

          infile
               Filename(s) of clm files

          Print header size of clm data file:
               headersize temp_1901-2003.clm

     EXIT STATUS
          headersize returns a zero exit status if file header of clm
          file can be read.  Non zero is returned in case of failure.
