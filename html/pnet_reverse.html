NAME
          pnet_reverse -  Reverse direction of all connections

     SYNOPSIS
          #include <mpi.h>
          #include "pnet.h"

          int pnet_reverse(Pnet *pnet);

     DESCRIPTION
          Function reverses direction of all connections.

          Connection have to be added before by calls of
          pnet_addconnect. Function must be called before pnet_setup.
          The argument of pnet_reverse is:

          pnet Pointer to initialized Pnet struct

     RETURN VALUE
          Upon successful completion PNET_OK is returned.

     ERRORS
          PNET_ALLOC_ERR
               Cannot allocate memory.
