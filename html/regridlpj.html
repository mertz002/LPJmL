     NAME
          regridlpj - regrid CRU input data set for selected countries
          or specified grid file

     SYNOPSIS
          regridlpj [-dir path] filesuffix [countrycode ...]

     DESCRIPTION
          Program regrids CRU input files for selected countries if
          country codes are provided. If country code is missing grid
          file grid_filesuffix.clm is used for regridding.
          Configuration files input_filesuffix.conf and
          input_filesuffix.js are created in the current directory.

     OPTIONS
          -dir path
               set directory for regridded input files

          filesuffix
               File suffix for regridded input files and configuration
               file

          countrycode
               ISO 3166-1 alpha-3 abbreviations for countries or
               rountry codes as numbers defined in include/managepar.h

     EXAMPLES
          Create input data files for USA:
               regridlpj usa USA

          Create input file for amazon basin:
               regridlpj amazon

          grid file grid_amazon.clm in the current directory is used
          for regridding.

     EXIT STATUS
          Non zero is returned in case of failure.
