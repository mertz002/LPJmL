NAME
          pnet_strerror - Return string describing Pnet error

     SYNOPSIS
          #include <mpi.h>
          #include "pnet.h"

          char *pnet_strerror(void)

     DESCRIPTION
          Function returns pointer to string describing the Pnet error

     RETURN VALUE
          Upon successful completion error message string is returned.
