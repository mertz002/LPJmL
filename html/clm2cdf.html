     NAME
          clm2cdf - convert CLM files into NetCDF files

     SYNOPSIS
          clm2cdf [-global] [-scale factor] [-cellsize size] [-int]
          [-float] [-landuse] [-notime] [-compress level] [-units u]
          [-descr d] varname gridfile clmfile netcdffile

     DESCRIPTION
          Program converts CLM input data into NetCDF input data.

     OPTIONS
          -global
               use global grid for NetCDF file

          -scale factor
               scaling factor for CLM version 1 files. Default is one

          -cellsize size
               set cell size of grid file to size

          -int set data type in CLM file to int, default is short

          -float
               set data type in CLM file to float, default is short

          -landuse
               convert land-use input data

          -notime
               No time dimension in NetCDF file

          -compress level
               compression level for NetCDF4 files

          -units u
               set units in NetCDF file

          -descr d
               set long name in NetCDF file

          varname
               variable name in NetCDF file

          gridfile
               filename of grid data file

          clmfile
               filename of CLM data file

          netcdffile
               filename of NetCDF file created

     EXAMPLES
          Create data file for temperature:
               clm2cdf -units celsius -descr temperature temp grid.bin
               tmp.clm tmp.nc

     EXIT STATUS
          clm2cdf returns a zero exit status if the NetCDF file is
          created.  Non zero is returned in case of failure.
