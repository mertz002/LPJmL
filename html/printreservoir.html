     NAME
          printreservoir - Print contents of reservoir file for LPJmL

     SYNOPSIS
          printreservoir clmfile ...


     DESCRIPTION
          Program prints contents of reservoir file  stored in clm
          data files for LPJmL.

     OPTIONS
          clmfile filename of reservoir data file

     EXIT STATUS
          printreservoir return a zero exit status if clm file could
          be printed.  Non zero is returned in case of failure.
