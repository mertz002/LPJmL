     NAME
          lpj_paths.sh - Set environmental variables used by LPJmL

     SYNOPSIS
           .  lpj_paths.sh

     DESCRIPTION
          Script sets environmental variables used by LPJmL. The root
          directory is defined and the search path for commands and
          man pages are extended.  The call of
            . $LPJROOT/bin/lpj_paths.sh

          should be added to the ~./profile
