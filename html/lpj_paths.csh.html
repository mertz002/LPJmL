     NAME
          lpj_paths.csh - Set environmental variables used by LPJmL
          for C shell

     SYNOPSIS
           .  lpj_paths.sh

     DESCRIPTION
          Script sets environmental variables used by LPJmL. The root
          directory is defined and the search path for commands and
          man pages are extended.  The call of
            source $LPJROOT/bin/lpj_paths.csh

          should be added to the ~./cshrc  file.
