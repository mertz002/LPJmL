     NAME
          soil2cdf - convert binary files into NetCDF files

     SYNOPSIS
          soil2cdf [-global] [-cellsize size] [-compress level]
          [-descr d] varname gridfile soilfile netcdffile

     DESCRIPTION
          Program converts binary input files into NetCDF file.

     OPTIONS
          -global
               use global grid for NetCDF file

          -cellsize size
               set cell size in grid file to size

          -compress level
               compression level for NetCDF4 files

          -descr d
               set long name in NetCDF file

          varname
               variable name in NetCDF file

          gridfile
               filename of grid data file

          soilfile
               filename of binary soil data file

          netcdffile
               filename of NetCDF file created

     EXAMPLES
          Create data file for soil:
               soil2cdf soilcode grid.bin soil.bin soil.nc

     EXIT STATUS
          soil2cdf returns a zero exit status if the NetCDF file is
          created.  Non zero is returned in case of failure.
