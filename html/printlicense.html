NAME
          printlicense - Print LPJmL license file

     SYNOPSIS
          #include "lpj.h"

          void printlicense(void);


     DESCRIPTION
          Function prints LPJmL license file on stdout. Environment
          variable LPJROOT has to be set accordingly to find locaton
          of license file.

     RETURN VALUE
          None.
