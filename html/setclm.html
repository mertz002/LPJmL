     NAME
          setclm - set value in header of clm files for LPJmL

     SYNOPSIS
          setclm
          {id|version|order|firstyear|nyear|firstcell|ncell|nbands}
          value clmfile


     DESCRIPTION
          Program sets file header and data stored in clm data files
          for LPJmL.

     OPTIONS
          {id|version|order|firstyear|nyear|firstcell|ncell|nbands}
               Define record to be set

          value
               value written in clm file

          clmfile
               filename of clm data file

     EXAMPLE
          Set version of clm file to 2:
               setclm version 2 tmp.clm

     EXIT STATUS
          setclm return a zero exit status if value in clm file could
          be set.  Non zero is returned in case of failure.
