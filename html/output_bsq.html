     NAME
          output_bsq - Concatenates all output files from distributed
          LPJmL simulations.

     SYNOPSIS
          output_bsq [-outpath dir] [-inpath dir] [-nyear year]

     DESCRIPTION
          Script concatenates all output files created by the LPJmL
          model to one output file. It used to join output files from
          lpjdistribute and lpjdistribute_river scripts. Script is
          obsolete for the MPI version of lpjml.

     OPTIONS
          -outpath dir
               sets directory of the concatenated output files.
               Default is output.

          -inpath dir
               sets directory of the output files to be joined.
               Default is output.

          -nyear n
               Number of simulation years. Default is 103.

     EXIT STATUS
          output_bsq returns a zero exit status if the output files
          have been concatenated.  Non zero is returned in case of
          failure.
