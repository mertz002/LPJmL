NAME
          fprintconfig, printconfig - Print LPJmL configuration

     SYNOPSIS
          #include "lpj.h"

          void fprintconfig(FILE *file,const Config *config);
          void printconfig(const Config *config);


     DESCRIPTION
          Function prints LPJmL configuration. printconfig() writes
          output to standard output.  The arguments of fprintconfig
          are:

          file File pointer to output stream.

          config
               LPJmL configuration data. Must be read by readconfig().

     RETURN VALUE
          None.
